# DynamicIslandStandalone Module

Standalone loader for Dynamic Island, independent from `iiBar` visibility and hover policy.

## Type

- `DynamicIslandStandalone`

## Loader id

- `iiDynamicIslandStandalone`

## IPC target

- `dynamicIslandStandalone`

Available calls:

```bash
qs ipc call dynamicIslandStandalone toggle
qs ipc call dynamicIslandStandalone open
qs ipc call dynamicIslandStandalone close
qs ipc call dynamicIslandStandalone status
```
